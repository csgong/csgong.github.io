元素

html

html文档的根元素，可以通过document.documentElement获取到html元素。

特有属性:

  • lang 推荐html元素要指定lang属性
  • manifest manifest属性用于缓存

head

document的元数据集合，包含 base，link，meta，noscript，script，style，template，title，其中base和title不多于一个，可以通过document.head获取到head元素。

title

document的标题或名称,可以通过document.title来设置title

base

base用于指定document中所有的链接的默认browsing context。每个文档中不得包含多个 base 元素。

属性:

  • href 规定页面中所有相对链接的基准URL,这其中包括、、、标签中的URL
  • target 在何处打开页面中所有的链接,可选的值有_blank，_self，_top，_framename
<!DOCTYPE html>
<html lang="en">
    <head>
        <title>This is an example for the &lt;base&gt; element</title>
        <base href="https://www.example.com/news/index.html" target="_self">
    </head>
    <body>
        <p>link1 <a href="archives.html">link1</a>.</p>
        <p>link2 <a href="https://www.aaa.com">link2</a>.</p>
    </body>
</html>

上面例子中link1的链接地址是 “https://www.example.com/news/archives.html", link2由于是绝对地址，所以它的地址没有变化，还是https://www.aaa.com，link1和link2的打开方式都是_self。 _blank, _self, _parent, or _top

q & blockquote & cite

<q>（quotation）是inline元素，适用短引用，默认会渲染成带有“”的一小段文字

<blockquote>是block元素，适用长的引用。<blockquote>默认会想染成带有缩进的单独一段内容

<cite> 标签通常表示它所包含的文本对某个参考文献的引用，比如书籍或者杂志的标题。 按照惯例，引用的文本将以斜体显示,例如：

<blockquote cite="https://www.huxley.net/bnw/four.html">
    <p>Words can be like X-rays, if you use them properly – they'll go through anything. You read and you're pierced.</p>
</blockquote>
<cite>Aldous Huxley, Brave New World</cite>

列表ol & ul & li & dl & dt & dd

<ol>（ordered list）是有序列表，默认以数字形式排序

<ul>(unordered list)是无序列表，默认以 **.**标注列表项

<li>(list item) 是列表项，配合ul和ol使用。

ol的type属性用于指定配许形式，默认为1，可选的值还有A、a、I、i。

li元素可以通过type和value属性改变当前列表项的排序方式和排序值：

<ul>
    <li type="1" value="4">Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ul>
<ol type="a">
     <li type="1" value="4">Coffee</li>
     <li>Tea</li>
     <li>Milk</li>
</ol>

<dl> 是包含一个或多个term-descript(ion groups的列表, html5规范定义如下：Term-description groups may be names and definitions, questions and answers, categories and topics, or any other groups of term-description pairs.

<dl>
  <dt>Blanco tequila</dt>
  <dd>The purest form of the blue agave spirit...</dd>
  <dt>Reposado tequila</dt>
  <dd>Typically aged in wooden barrels for between two and eleven months...</dd>
</dl>
<dl>
  <dt>Authors:</dt>
  <dd>Remy Sharp</dd>
  <dd>Rich Clark</dd>
  <dt>Editor:</dt>
  <dd>Brandan Lennox</dd>
  <dt>Category:</dt>
  <dd>Comment</dd>
</dl>

Character Entities

HTML中<，>，&等有特殊含义，不能直接使用,如果想显示这些字符，需要使用字符实体（Character Entities）， 字符实体由 &entity_name; 或者 &#entity_number; 组成,常见的字符实体可以查看这份文档

em

<em> 用于语气上的强调，也就是我们说话时重音所在的内容，重音放在不同地方，表达的意思可能完全不一样。

strong

<strong>用于表示内容的重要性、严肃性(警告、通知)、紧急性

<h1>Chapter 1: <strong>The Praxis</strong></h1>
<strong>Avoid the ducks.</strong> Take any gold you find.
<p><strong>Warning.</strong> This dungeon is dangerous.

i

<i>用于定义与当前内容不同的部分，如术语、名言和其他特有名称

<p>He named his car <i>The lightning</i>, because it was very fast.</p>
<p>The Latin phrase <i class="latin">Veni, vidi, vici</i> is often
mentioned in music, art, and literature</p>

b

<b>用于吸引注意力，不带有情绪，声音等额外的重要性

The following example shows a use of the b element to highlight key words without marking them up as important:

<p>The <b>frobonitor</b> and <b>barbinator</b> components are fried.</p>

pre

<pre>元素表示预定义格式文本。在该元素中的文本通常按照原文件中的编排，以等宽字体的形式展现出来，文本中的空白符（比如空格和换行符）都会显示出来

<p>This is the <code>Panel</code> constructor:</p>
<pre><code>function Panel(element, canClose, closeHandler) {
  this.element = element;
  this.canClose = canClose;
  this.closeHandler = function () { if (closeHandler) closeHandler() };
}</code></pre>

code

<code>代表可以被计算机识别的任何代码的片段

<p>The <code>code</code> element represents a fragment of computer
code.</p>

<p>When you call the <code>activate()</code> method on the
<code>robotSnowman</code> object, the eyes glow.</p>
